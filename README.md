# AGX

Adaptive Graph X is a dynamic graph system which learns and adapts in real time and optimizing
path selection. This is done by using self-learning and predictive evaluation for smarter and efficient
decision-making.

FEATURES:
- Dynamic Graph Structure
- BFS / DFS Auto Switching
- Predictive Path Selection
- Adaptive Learning Engine
- Intent-Based Computing Simulation
- Memory Optimization
- Runtime Graph Evolution
- Node Activity Tracking
- Duplicate Removal
- Graph Compression
- Weighted + Unweighted Graph Support
- Directed + Undirected Graph Support
- Self-Learning Traversal Optimization
